# financial-analysis

*Financial Analysis of Countries by Year
* Overview
This project analyzes the financial performance of countries over multiple years.
The data is visualized using interactive dashboards to better understand trends, comparisons, and geographical distributions.

* Key Features
Map Visualization:** Displays the geographical distribution of financial data by country
Column Chart:** Compares financial metrics across different years
Line Chart:** Shows trends and changes in financial performance over time

* Tools & Technologies
Power BI
Microsoft Excel

* Dataset
The dataset includes country-based financial data organized by years.
It has been cleaned and structured in Excel before being imported into Power BI.
